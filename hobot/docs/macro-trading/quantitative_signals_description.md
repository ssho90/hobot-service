# 정량 시그널 API 파라미터 설명

이 문서는 `/api/macro-trading/quantitative-signals` API 응답값의 각 파라미터에 대한 상세 설명입니다. LLM이 거시경제 시그널을 분석할 때 참고할 수 있도록 작성되었습니다.

## API 응답 구조

```json
{
  "status": "success",
  "timestamp": "2025-11-22 19:02:13",
  "signals": { ... },
  "parameters": { ... }
}
```

---

## signals.yield_curve_spread_trend (장단기 금리차 추세 추종 전략)

금리 곡선의 장단기 금리차(스프레드)를 분석하여 경기 사이클과 금리 정책 방향을 판단하는 지표입니다.

### 기본 지표

- **spread** (float, %)
  - **의미**: 현재 장단기 금리차
  - **계산식**: DGS10 (10년물 국채 금리) - DGS2 (2년물 국채 금리)
  - **해석**: 
    - 양수: 정상적인 금리 곡선 (장기 금리 > 단기 금리)
    - 음수: 역전된 금리 곡선 (경기 침체 신호)
    - 값이 클수록 스프레드가 넓음 (경기 확장 기대)
    - 값이 작을수록 스프레드가 좁음 (경기 둔화 신호)

- **spread_fast** (float, %)
  - **의미**: 스프레드의 단기 이동평균
  - **계산**: 스프레드의 20일 이동평균
  - **용도**: 단기 추세 파악

- **spread_slow** (float, %)
  - **의미**: 스프레드의 장기 이동평균
  - **계산**: 스프레드의 120일 이동평균
  - **용도**: 장기 추세 파악

- **yield_trend** (float, %)
  - **의미**: 10년물 국채 금리(DGS10)의 장기 이동평균
  - **계산**: DGS10의 200일 이동평균
  - **용도**: 금리 대세 판단 기준선

- **current_dgs10** (float, %)
  - **의미**: 현재 10년물 미국 국채 금리
  - **해석**: 장기 금리 수준, 경기 전망을 반영

- **current_dgs2** (float, %)
  - **의미**: 현재 2년물 미국 국채 금리
  - **해석**: 단기 금리 수준, 연준 정책 금리를 반영

### 추세 지표

- **spread_trend** (string)
  - **가능한 값**: "Steepening" 또는 "Flattening"
  - **의미**: 스프레드 추세 방향
  - **판단 기준**: spread_fast와 spread_slow 비교
    - "Steepening": spread_fast > spread_slow (스프레드 확대 추세)
    - "Flattening": spread_fast < spread_slow (스프레드 축소 추세)

- **spread_trend_kr** (string)
  - **의미**: spread_trend의 한글 번역
  - **가능한 값**: "확대" 또는 "축소"

- **yield_regime** (string)
  - **가능한 값**: "Rising" 또는 "Falling"
  - **의미**: 금리 대세 방향
  - **판단 기준**: current_dgs10과 yield_trend 비교
    - "Rising": current_dgs10 > yield_trend (금리 상승 추세)
    - "Falling": current_dgs10 < yield_trend (금리 하락 추세)

- **yield_regime_kr** (string)
  - **의미**: yield_regime의 한글 번역
  - **가능한 값**: "금리 상승 추세" 또는 "금리 하락 추세"

### 국면 판단

- **regime** (string)
  - **의미**: 금리 곡선과 금리 대세를 결합한 4가지 국면
  - **가능한 값**:
    - "Bull Steepening": 스프레드 확대 + 금리 하락 (경기 부양 기대)
    - "Bear Steepening": 스프레드 확대 + 금리 상승 (인플레/재정 공포)
    - "Bull Flattening": 스프레드 축소 + 금리 하락 (경기 둔화 초기)
    - "Bear Flattening": 스프레드 축소 + 금리 상승 (강력한 긴축)

- **regime_kr** (string)
  - **의미**: regime의 한글 번역 및 해석
  - **예시**: "Bull Flattening (경기 둔화 초기)"

## signals.real_interest_rate (실질 금리)

- **타입**: float (%)
- **의미**: 명목 금리에서 인플레이션을 차감한 실질 금리
- **계산식**: DGS10 (명목 금리) - CPI 증가율 (연율화)
- **해석**:
  - 양수: 실질 금리가 양수 → 통화 정책이 경기 과열 억제 효과
  - 음수: 실질 금리가 음수 → 통화 정책이 완화적, 자산 가격 상승 압력
  - 값이 클수록 통화 긴축 강도가 높음
  - 값이 작을수록 통화 완화 강도가 높음

## signals.taylor_rule_signal (테일러 준칙 신호)

- **타입**: float (%)
- **의미**: 테일러 준칙에 따른 이론적 목표 금리와 현재 금리의 차이
- **계산식**: Target_Rate - FEDFUNDS (현재 연준 금리)
  - Target_Rate = r* + π + 0.5(π - π*) + 0.5(y - y*)
    - r*: 자연 이자율 (기본값 2.0%)
    - π: 현재 PCE 인플레이션율 (연율화)
    - π*: 목표 인플레이션율 (기본값 2.0%)
    - y - y*: GDP 갭 (현재는 0으로 가정)
- **해석**:
  - 양수: 목표 금리 > 현재 금리 → 금리 인상 여지 존재, 통화 긴축 필요
  - 음수: 목표 금리 < 현재 금리 → 금리 인하 여지 존재, 통화 완화 가능
  - 절댓값이 클수록 정책 조정 필요성 높음
  - 0에 가까울수록 현재 금리 수준이 적정

## signals.net_liquidity (연준 순유동성)

연준의 자산에서 재무부 계정과 역레포를 차감한 실제 시장에 공급되는 유동성을 나타냅니다.

- **net_liquidity** (float, Millions of Dollars)
  - **의미**: 연준 순유동성 절대값
  - **계산식**: WALCL (연준 자산) - WTREGEN (재무부 일반계정) - RRPONTSYD (역레포)
  - **해석**:
    - 값이 클수록 시장 유동성이 풍부
    - 값이 작을수록 시장 유동성이 부족
    - 증가 추세: 유동성 공급 확대 → 자산 가격 상승 압력
    - 감소 추세: 유동성 공급 축소 → 자산 가격 하락 압력

- **ma_trend** (float)
  - **의미**: 순유동성 이동평균의 추세 방향
  - **가능한 값**:
    - `1`: 상승 추세 (이동평균이 상승 중)
    - `-1`: 하락 추세 (이동평균이 하락 중)
    - `0`: 보합 (이동평균 변화 없음)
  - **해석**:
    - 1: 유동성 공급 확대 중 → 자산 매수 신호
    - -1: 유동성 공급 축소 중 → 자산 매도 신호

- **ma_value** (float, Millions of Dollars)
  - **의미**: 순유동성의 이동평균값
  - **계산**: 4주(28일) 이동평균
  - **용도**: 단기 변동성을 제거한 추세 파악

## signals.high_yield_spread (하이일드 스프레드)

하이일드(투자부적격) 채권과 국채 간 금리 차이로, 시장의 위험 선호도와 유동성 상태를 나타냅니다.

- **spread** (float, %)
  - **의미**: 하이일드 채권 스프레드
  - **해석**:
    - 값이 작을수록: 위험 선호도 높음, 유동성 풍부
    - 값이 클수록: 위험 회피 심리, 유동성 경색
    - 3.5% 미만: 매우 낮은 수준 (유동성 매우 풍부)
    - 5.0% 이상: 경계 수준 (유동성 경색 시작)
    - 10.0% 이상: 위기 수준 (금융 위기)

- **signal** (float)
  - **의미**: 유동성 상태 신호
  - **가능한 값**:
    - `1`: Greed (탐욕) - 유동성 매우 풍부, 주식 적극 매수 신호
    - `0`: Fear/Neutral (공포/중립) - 유동성 경색 시작, 주식 비중 축소 고려
    - `-1`: Panic (공황) - 금융 위기, 전량 현금/달러/국채 전환 고려

- **signal_name** (string)
  - **의미**: signal의 이름
  - **가능한 값**: "Greed", "Fear", "Neutral", "Panic"

- **week_change** (float, %)
  - **의미**: 전주 대비 스프레드 변화율
  - **계산**: (현재 스프레드 / 7일 전 스프레드 - 1) × 100
  - **해석**:
    - 양수: 스프레드 확대 (위험 회피 심리 증가)
    - 음수: 스프레드 축소 (위험 선호도 증가)
    - 절댓값이 클수록 변화가 급격함

## signals.additional_indicators (추가 지표)

- **타입**: object (dictionary)
- **의미**: 실업률, 고용 지표 등 추가 거시경제 지표
- **현재 포함 가능한 지표**:
  - `unemployment_rate`: 실업률 (%)
  - `payroll_growth`: 비농업 고용 전월 대비 증가율 (%)
- **참고**: 데이터가 없으면 빈 객체 `{}`로 반환됨

## parameters (계산 파라미터)

시그널 계산에 사용된 파라미터 값입니다.

- **natural_rate** (float, %)
  - **의미**: 자연 이자율 (r*)
  - **용도**: 테일러 준칙 계산에 사용
  - **기본값**: 2.0%
  - **해석**: 경기와 인플레이션이 안정적일 때의 이자율 수준

- **target_inflation** (float, %)
  - **의미**: 목표 인플레이션율 (π*)
  - **용도**: 테일러 준칙 계산에 사용
  - **기본값**: 2.0%
  - **해석**: 연준이 목표로 하는 인플레이션율

- **liquidity_ma_weeks** (integer | null)
  - **의미**: 순유동성 이동평균 기간 (주)
  - **기본값**: null (설정 파일에서 가져옴, 일반적으로 4주)
  - **용도**: net_liquidity의 ma_trend와 ma_value 계산에 사용

## 종합 분석 가이드

### 1. 금리 곡선 분석
- `yield_curve_spread_trend.regime`을 확인하여 현재 경기 사이클 국면 파악
- Bull Flattening: 경기 둔화 초기 → 방어적 자산 배분 고려
- Bear Steepening: 인플레 우려 → 인플레 헤지 자산 고려

### 2. 통화 정책 분석
- `taylor_rule_signal`: 양수면 금리 인상 압력, 음수면 금리 인하 압력
- `real_interest_rate`: 실질 금리 수준으로 통화 정책 강도 파악

### 3. 유동성 분석
- `net_liquidity.ma_trend`: -1이면 유동성 축소 중 → 자산 가격 하락 압력
- `high_yield_spread.signal`: 1이면 유동성 풍부 → 자산 매수 기회

### 4. 시장 심리 분석
- `high_yield_spread.spread`: 값이 작을수록 위험 선호도 높음
- `high_yield_spread.week_change`: 급격한 변화는 시장 변동성 증가 신호

### 5. 모순 신호 확인
- 예: `yield_curve_spread_trend.regime`이 "Bull Flattening" (경기 둔화)인데 `high_yield_spread.signal`이 1 (Greed)인 경우
- 이런 모순은 전환기 신호일 수 있으므로 주의 깊게 관찰 필요

## 주의사항

1. **데이터 지연**: 일부 지표(특히 PCEPI)는 최신 데이터가 3개월 전 것일 수 있음
2. **상대적 해석**: 절대값보다는 추세와 변화율에 주목
3. **종합 판단**: 단일 지표보다 여러 지표를 종합하여 분석
4. **시장 맥락**: 지표만으로 판단하지 말고 현재 시장 상황과 함께 고려

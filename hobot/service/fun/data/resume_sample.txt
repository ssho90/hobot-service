[Line 직무 질문 두가지]
1. 대표적인 프로젝트 한 개 이상, 각각에 대해서 1) 진행 방식, 2) 본인의 역할, 3) 문제 해결 경험을 포함하여 구체적으로 설명해 주세요.

자사 솔루션에 AI 기능을 도입한 프로젝트가 저의 대표적인 AI 프로젝트입니다. 프로젝트 내에 AI 전문 인력이 없었기에, 제가 주도적으로 AI 기능 도입 과제를 맡아 진행했습니다. 가장 큰 고민은 'AI 프로젝트 경험이 없는 상태에서 어떻게 비용 효율적으로 POC 할 것 인가' 였습니다. 고객이나 저나 AI 프로젝트를 진행해 본 경험이 없었기 때문에, 최대한 빠르게 POC를 진행해서 구현 가능성을 검토해야 했습니다. 그래서 Framework로는 Langchain을 선택했고, Vector DB로는 대시보드 모니터링용으로 사용중이었던 Elastic Search를 활용하여 인프라 구축 비용과 시간을 절약하는 방향으로 결정했습니다. LLM과 임베딩 모델은 다양한 모델을 테스트해보며 선별했고, 최종적으로 Gemini와 multilingual-e5 모델을 선정했습니다. 한 달 간의 POC 끝에 Langchain을 활용한 RAG 아키텍처 구축의 가능성을 프로젝트 구성원들에게 입증했고, Langchain을 활용해 고객에게 AI 서비스를 제공하는 것을 표준으로 잡고 어플리케이션 개발과 인프라 구축을 병행했습니다.

구현 단계에서는 임베딩 처리 시간이 예상보다 길다는 기술적 문제에 직면했습니다. 사용자가 질문에 대한 빠른 응답을 위해, 임베딩 파이프라인과 Vector DB 검색 및 LLM 호출 파이프라인을 분리하는 아키텍처를 설계했습니다. 임베딩 프로세스가 Batch와 GCP PubSub을 이용해 사용자 질문 전에 미리 수행되도록 했고, 사용자가 API 사용시 임베딩 과정을 거치지 않고 미리 Vector DB에 저장된 데이터를 조회하여 LLM을 호출하도록 구성하여 RAG의 응답속도를 개선할 수 있었습니다.

Langchain 어플리케이션 개발은 혼자 주도적으로 진행했으며, RAG 아키텍처 설계 및 구성, 어플리케이션 개발, 표준 가이드라인 작성은 동료 아키텍트들과 함께 수행했고 프로젝트 전반에 걸쳐 핵심적인 역할을 수행했습니다.


2. 도전적인 목표를 설정하여 주도적으로 추진해 본 경험을 공유해 주세요.

‘Azure Landing Zone 및 보안 RAG 아키텍처 구축' 프로젝트는 저에게 도전적인 목표를 설정하고 주도적으로 역량을 확장하는 중요한 경험이었습니다.
  당시 저는 주로 백엔드 개발 및 DevOps 아키텍트로서의 경험을 보유하고 있었으나, 해당 프로젝트는 클라우드 네트워크 및 보안 아키텍처 설계/구축 역량이 핵심인 인프라 아키텍트(Infra Architect) 역할을 요구했습니다. 저에게는 부족한 영역이었지만, 클라우드 인프라 전반, 특히 네트워크와 보안 분야의 전문성을 깊게 쌓고 싶다는 생각으로 지원을 했습니다.
  약 1억 원 규모의 프로젝트에 단독으로 투입되어, 초기 아키텍처 설계부터 VNet, Subnet, AGW 구성, WAF/Firewall/Private Endpoint를 활용한 보안 아키텍처 구성까지 전 과정을 혼자 책임져야 했습니다. 예상대로 과정은 순탄치 않았습니다. 초기에는 고객의 기술 질문에 즉각적으로 답변하지 못해 당황하기도 했고, 특히 모든 Azure 자원이 Private Networking되도록 Private Endpoint를 연계하는 과정에서 기술적인 난관에 부딪히기도 했습니다. 하지만 저는 프로젝트 완수를 위해 많은 노력들을 했습니다. 매일 퇴근 후 Azure 공식 문서와 기술 블로그를 보며 부족한 학습했고, 혼자 해결하기 어려운 부분은 동료들에게 제가 부딪힌 문제 상황과 고민을 구체적으로 설명하며 조언을 구했습니다. 고객과의 관계에 있어서 모르는 부분은 솔직하게 인정하되, 학습을 통해 빠른 시간 내에 해결 방안을 찾아 제시하며 고객과의 신뢰를 구축하는 데 집중했습니다.
  이러한 노력 끝에, 프로젝트를 성공적으로 완수하며 Azure 보안 RAG 아키텍처를 구축할 수 있었습니다. 이 경험은 단순히 프로젝트 성공을 넘어, 제 기술 역량을 클라우드 인프라 아키텍처 영역까지 확장하는 결정적인 계기가 되었습니다. LINE Plus에서 AI Agent Platform을 구축하고 고도화하는 과정에 적극적으로 도전하고 싶습니다.

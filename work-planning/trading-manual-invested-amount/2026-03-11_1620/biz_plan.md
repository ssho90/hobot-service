# Trading Dashboard 수동 투자원금 입력 계획서

## 목적

`/trading` 화면의 투자원금 카드에서 사용자가 금액을 직접 입력할 수 있게 하고, 카드 우측 상단 아이콘으로 편집을 열도록 만든다.

## 범위

- `hobot-ui-v2/src/components/TradingDashboard.tsx`
- 투자원금 카드의 UI/상태 처리
- 사용자별 로컬 저장

## 완료 조건

1. 투자원금 카드 우측 상단에 편집 아이콘이 노출된다.
2. 아이콘 클릭 시 금액 입력 UI가 열린다.
3. 저장한 값은 새로고침 후에도 유지된다.
4. 초기화로 기본 계산값으로 되돌릴 수 있다.

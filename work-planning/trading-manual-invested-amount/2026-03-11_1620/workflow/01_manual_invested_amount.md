# Session 01. 투자원금 직접 입력 UI

## 목표

- `/trading`의 투자원금 카드에서 금액을 직접 입력할 수 있게 한다.

## 진행 기록

1. 기존 투자원금 카드는 백엔드의 `net_invested_amount`가 없으면 `총 평가금액 - 총 평가손익`으로만 계산 표시했다.
2. 카드 우측 상단에 편집 아이콘과 초기화 아이콘을 배치했다.
3. 클릭 시 카드 내부에서 바로 금액을 입력하고 저장/취소할 수 있게 했다.
4. 입력값은 사용자 ID 기준 `localStorage`에 저장되도록 했다.
5. `npm run build`를 통해 타입/번들 검증을 통과했다.
6. 후속 피드백 반영으로 `수동 입력값 적용중` 문구를 제거하고, 수동 투자원금을 기준으로 `평가손익`과 `수익률` 카드가 재계산되도록 수정했다.

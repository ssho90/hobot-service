# KR 거시/부동산 자동화 계획

## 목표
- 누락된 KR 거시 수집과 KR 부동산 수집 파이프라인을 스케줄러에 등록한다.
- 기존 통합 스케줄러(`start_all_schedulers`) 실행 시 자동 포함되도록 연결한다.

## 작업 항목
1. `scheduler.py`에 env 기반 실행 함수 추가
- KR 거시 수집 실행 함수
- KR 부동산 수집/집계/그래프 동기화 실행 함수

2. `scheduler.py`에 스케줄 등록 함수 추가
- KR 거시 일간 스케줄
- KR 부동산 일간 스케줄

3. 통합 스케줄러 연결
- `start_all_schedulers()`에 신규 setup 함수 호출 추가

4. 검증
- 문법 컴파일 확인
- 실제 스케줄 등록 결과 확인

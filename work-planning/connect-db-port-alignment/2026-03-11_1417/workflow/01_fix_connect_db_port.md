# Session 01. connect_db 포트 정렬

## 목표

- `connect_db.sh`의 로컬 터널 포트를 백엔드 DB 설정과 맞춘다.

## 진행 기록

1. `connect_db.sh`는 `LOCAL_PORT=3306`으로 터널을 열고 있었다.
2. `hobot/.env`는 `DB_PORT=3307`을 사용하고 있었다.
3. 스크립트를 `3307 -> 3306` 포트 포워딩으로 수정했다.
4. 이제 앱은 `127.0.0.1:3307`로 접속하고, SSH 터널이 그 요청을 EC2 MySQL `3306`으로 전달한다.
